# 🐺 Читавук

**Читавук** (читать + *вук* — «волк») — приложение для изучения сербского языка через
чтение. Открываешь текст (PDF/DOCX), тапаешь по слову — получаешь перевод, начальную
форму и грамматический разбор; сохраняешь слова в карточки для интервального повторения.
Маскот — умный волк Читавук.

## Возможности

- 📖 **Читалка** PDF/DOCX с настройками (шрифт, размер, фон любого цвета, выравнивание,
  «диагональное»/bionic выделение основы слова).
- 🔤 **Разбор слова по тапу:** перевод (онлайн), лемма, часть речи, словоформы;
  авто-починка «битых» сербских букв (š/č/ć/ž/đ).
- 🧠 **Грамматика «Почему так?»:** падежи (склонение) и три времени (prezent/perfekat/
  futur), интерактивная шпаргалка по падежам, грамматика в виде карточек.
- 🗂 **Библиотека:** папки-коллекции, переименование книг, прогресс чтения.
- 🃏 **Карточки (SRS, SM-2):** цвет по сложности, подсказки про ассоциации/эмоции.
- 🎨 Сербская визуальная идентичность (палитра вышивки, орнамент, маскот-волк).

## Технологии

- **Frontend:** Flutter (Windows/Android), provider, sqflite, syncfusion PDF.
- **Backend (опционально):** Python + FastAPI + CLASSLA (морфология сербского).
- **Словарь:** собран из трибанка UD_Serbian-SET (формы, леммы, признаки UD).

## Быстрый старт

```bash
# приложение (Windows)
cd frontend
flutter pub get
flutter run -d windows

# бэкенд (опционально, для контекстного разбора)
cd backend
pip install -r requirements.txt
python main.py            # http://localhost:8000
```

Приложение работает и без бэкенда: морфология — из встроенного словаря, перевод —
по интернету напрямую.
